# CoderDojo Hommachi Pages

## Installation

```
$ git clone https://github.com/coderdojo-hommachi/coderdojo-hommachi.github.io.git
$ cd coderdojo-hommachi.github.io/
$ npm setup
$ npm start
```

## Usage

* Open server address:

```
$ open http://127.0.0.1:4000/
```

## Requirements

* [Jekyll](https://jekyllrb.com/)
