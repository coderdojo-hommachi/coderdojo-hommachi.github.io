# CoderDojo Osakasayama / Hommachi Pages

## Installation

```
$ git clone https://github.com/coderdojo-hommachi/coderdojo-hommachi.github.io.git
$ cd coderdojo-hommachi.github.io/
$ npm start
```

## Usage

- Open server address:

```
$ open http://127.0.0.1:8000/
```

## Requirements

- [Gatsby](https://www.gatsbyjs.org/)
